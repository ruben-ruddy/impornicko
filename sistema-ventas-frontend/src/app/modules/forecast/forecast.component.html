Promedio Móvil:

Suaviza las fluctuaciones usando el promedio de períodos anteriores. Bueno para datos con variabilidad.

  • Suavizado exponencial con factor α
  • Ideal para datos con ruido
  • Respuesta rápida a cambios recientes

Modelo Estacional:

  • Óptimo para ventas con temporadas
