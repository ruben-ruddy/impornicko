Regresión Lineal:

Método estadístico que modela la relación entre variables mediante una línea recta. Ideal para tendencias consistentes.

  • Mejor para datos con tendencia lineal clara
  • Calcula coeficiente de determinación (R²)
  • Proporciona intervalos de confianza
} @if (forecastRequest.metodo === 'promedio_movil') {

Promedio Móvil:

Suaviza las fluctuaciones usando el promedio de períodos anteriores. Bueno para datos con variabilidad.

  • Suavizado exponencial con factor α
  • Ideal para datos con ruido
  • Respuesta rápida a cambios recientes
} @if (forecastRequest.metodo === 'estacional') {

Modelo Estacional:

  • Ajusta por factores estacionales
  • Óptimo para ventas con temporadas
